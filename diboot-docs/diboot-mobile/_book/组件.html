1. 组件

  • 图片上传
  • 图片预览
  • 关联数据列表选择
  • 列表分页组件
  • diboot-mobile内置了一些基础的功能组件，您也可以根据自己的业务来创建相关的功能组件。
  • 内置的基础组件在src/components/common文件夹下，并根据相关功能场景以文件夹做了区分。
    • image 文件夹下为图片上传以及图片预览相关的功能组件。
    • infinite 为列表页滚动加载数据的组件，提供了加载数据上拉翻页等功能。
    • select 为关联数据列表选择组件，多用于表单中选择关联数据使用。
